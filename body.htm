Para que serve o <body>?

O elemento <body> contém todo o conteúdo visível da página: textos, imagens, links, tabelas, formulários, etc. Em HTML antigo era comum colocar atributos no <body> para controlar cores. Hoje, isso faz-se de forma correta com CSS.

O que era usado “antigamente” (obsoleto)

Atributos como bgcolor, text, link, vlink e alink no <body> estão obsoletos e não devem ser usados em projetos atuais.

<body bgcolor="#FF8080" text="#004080" link="#00FF80" vlink="#008000" alink="#FF8000"> ... </body>

Como se faz hoje (recomendado)

Controla o fundo, a cor do texto e os estados dos links com CSS: :link, :visited, :hover e :active.

/* Fundo e texto da página */ body{ background: #ffe4e6; color: #0f172a; font-family: system-ui, Arial, sans-serif; } /* Estados de links */ a:link{ color: #0ea5e9; } a:visited{ color: #7c3aed; } a:hover{ color: #0284c7; } a:active{ color: #ea580c; }
Dica atual: se queres controlar cores em todo o site, usa variáveis CSS (:root { --cor: ... }) e aplica-as no body e nos links. Fica consistente e fácil de manter.
