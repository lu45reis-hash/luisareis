Listas Numeradas

As listas numeradas são usadas quando a ordem dos elementos é importante (por exemplo: passos de um processo, ranking, sequência de ações).

Em HTML, este tipo de lista é criado com a tag <ol> (ordered list) e cada item com <li>.

Exemplo básico

<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
  <li>Terceiro item</li>
</ol>

Resultado:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Listas hierárquicas (listas dentro de listas)

É possível misturar listas numeradas com não numeradas para criar estruturas hierárquicas:

<ol>
  <li>Planeamento
    <ul>
      <li>Recolha de requisitos</li>
      <li>Desenho da página</li>
    </ul>
  </li>
  <li>Desenvolvimento</li>
  <li>Testes</li>
</ol>

Resultado:

  1. Planeamento
    • Recolha de requisitos
    • Desenho da página
  2. Desenvolvimento
  3. Testes

Dica: podes controlar o tipo de numeração (1, a, i, A, I...) usando CSS com a propriedade list-style-type.
