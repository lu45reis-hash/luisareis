Além do <input>, dois elementos muito importantes em formulários são: <select> (lista de opções) e <textarea> (texto com várias linhas).

Boa prática: usa <label> ligado ao campo através de for e id para melhorar a acessibilidade.

1) Lista de opções – <select> + <option>

O <select> cria uma lista de seleção. Cada opção é criada com <option>.

Código

<label for="browser">Browser preferido</label>
<select id="browser" name="browser" required>
  <option value="" selected disabled>Escolhe uma opção…</option>
  <option value="chrome">Google Chrome</option>
  <option value="firefox">Mozilla Firefox</option>
  <option value="edge">Microsoft Edge</option>
  <option value="safari">Safari</option>
</select>

Resultado

required obriga a escolher uma opção válida.

Mostrar várias opções e permitir seleção múltipla

Com size defines quantas opções ficam visíveis. Com multiple permites escolher mais do que uma.

Código (size + multiple)

<label for="langs">Linguagens que queres aprender</label>
<select id="langs" name="langs" size="4" multiple>
  <option value="html">HTML</option>
  <option value="css">CSS</option>
  <option value="js">JavaScript</option>
  <option value="py">Python</option>
</select>

Resultado

No computador: Ctrl/Shift para múltiplas opções.
Tem atenção que em telemóveis, a seleção múltipla pode ter comportamento diferente. Para múltiplas escolhas, muitas vezes é melhor usar checkbox.

2) Texto longo – <textarea>

O <textarea> é usado para mensagens ou comentários. Em HTML5 podes usar placeholder, maxlength e required.

Código

<label for="comentario">Comentário</label>
<textarea
  id="comentario"
  name="comentario"
  rows="5"
  placeholder="Escreve aqui a tua mensagem…"
  maxlength="300"
  required
></textarea>

Resultado

Dica: maxlength limita caracteres.
Resumo:
  • <select> → escolhas numa lista
  • <option> → cada opção
  • <textarea> → texto com várias linhas

Um bom próximo passo é criar um formulário completo (ex.: contacto/pesquisa) com validação HTML5 e layout organizado.
