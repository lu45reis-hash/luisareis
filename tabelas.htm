As tabelas em HTML servem para apresentar dados tabulares (linhas e colunas), como horários, listas de preços ou resultados.
Não devem ser usadas para layout — isso faz-se com CSS.

Estrutura básica de uma tabela

  • <table> — cria a tabela
  • <tr> — cria uma linha (table row)
  • <th> — célula de cabeçalho
  • <td> — célula de dados

Exemplo de código

<table>
  <tr>
    <th>Nome</th>
    <th>Idade</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>16</td>
  </tr>
</table>

Resultado visual

Nome Idade
Ana 16
Boa prática: usa CSS para bordas, cores e espaçamento. O atributo border="1" ainda funciona, mas é considerado antigo.
