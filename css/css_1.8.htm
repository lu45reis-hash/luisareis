Capítulo 2 · CSS e HTML moderno

Novos atributos: style, class e id

Estes 3 atributos continuam a ser fundamentais hoje. A diferença é que agora damos prioridade a CSS externo e a boas práticas: reutilização, manutenção, acessibilidade e consistência.

style (CSS inline)

Aplica estilo diretamente num elemento. É útil para testes rápidos, mas em excesso torna o projeto difícil de manter. 

<h3 style="font-weight:800; text-transform:uppercase; color:green;">
  O elemento H3 fica verde, em maiúsculas e em negrito
</h3>

O elemento H3 fica verde, em maiúsculas e em negrito

Correção importante: em CSS o certo é text-transform (não “font-transform”).

class (estilo reutilizável)

A class permite aplicar o mesmo estilo a vários elementos. É a forma mais comum de organizar estilos em sites modernos. 

<style>
  .classe1 { font-style: italic; color: darkblue; }
  .classe2 { font-weight: 800; }
  .classe3 { color: #d11; }
</style>

<h2 class="classe1">Texto em itálico e azul escuro</h2>
<div class="classe2">Texto em negrito</div>
<p class="classe3">Texto vermelho</p>

Texto em itálico e azul escuro

Texto em negrito

Texto vermelho

Um elemento pode ter várias classes: class="card destaque".

id (identificador único)

O id identifica um único elemento na página. No CSS, seleciona-se com #nome. 

<style>
  .geral { font-size: 1.15rem; font-style: italic; color: darkblue; }
  #a1 { color: #d11; }
</style>

<p class="geral">Texto grande azul escuro</p>
<p class="geral" id="a1">Texto grande vermelho</p>

Texto grande azul escuro

Texto grande vermelho

Boas práticas: evita usar id para “estilo geral” — usa class. O id é ótimo para âncoras (#secao) e JS.

Resumo rápido

  • style: rápido, mas difícil de manter se usado em excesso.
  • class: reutilizável, ideal para design consistente.
  • id: único na página, útil para âncoras e scripts.

Em projetos modernos, tenta manter a maior parte do CSS em ficheiros externos.
