O que é CSS

CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) é a linguagem usada para definir o aspeto visual das páginas web: cores, fontes, espaçamento, layout e até animações. Enquanto o HTML organiza o conteúdo, o CSS trata da apresentação.

A ideia de “cascata” significa que podem existir várias regras para o mesmo elemento e o browser decide qual aplicar com base em prioridades (origem da regra, especificidade e ordem).

- Hoje em dia normalmente dizemos apenas “CSS”. O “CSS1” foi a primeira especificação, mas o CSS evoluiu por módulos (ex.: cores, layout, flexbox, grid, etc.). 
/* Exemplo simples de CSS */
body {
  font-family: system-ui, Arial, sans-serif;
  color: #0f172a;
}

h1 {
  color: #1d4ed8;
}

.card {
  background: white;
  border-radius: 16px;
  padding: 16px;
}
