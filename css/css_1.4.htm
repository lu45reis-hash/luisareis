Navegadores que suportam CSS

Compatibilidade · Web moderna

Hoje em dia, o suporte a CSS é muito bom nos principais navegadores: Chrome, Edge, Firefox e Safari. A maior parte das propriedades comuns (tipografia, cores, Flexbox, Grid, animações, etc.) funciona de forma consistente.

- Regra prática atual: se testares em Chrome/Edge + Firefox + Safari, já cobres praticamente todos os utilizadores.

O que ainda pode variar

  • Funcionalidades muito recentes podem demorar a chegar a todos os navegadores. Nestes casos usa “fallbacks” (uma alternativa simples) ou verifica suporte antes de usar.
  • Diferenças no Safari (iOS) acontecem mais frequentemente em features novas, por isso convém testar em telemóvel.
  • Prefixos antigos (ex.: -webkit-) hoje são raros, mas ainda aparecem em algumas propriedades específicas.
Boa prática: escreve CSS com “progressive enhancement”. Ou seja: começa por um layout simples que funciona sempre e adiciona melhorias quando houver suporte.

Dica rápida de compatibilidade

Quando tiveres dúvidas sobre uma propriedade, procura sempre: suporte do browser + alternativa (“fallback”). Exemplos: usar display: flex com uma estrutura que ainda funciona em bloco, ou definir uma cor de fundo antes de um gradiente.
