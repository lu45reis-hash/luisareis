Novos elementos: <style>, <link> e <span>

Hoje, o CSS pode estar dentro da página (estilo interno), num ficheiro externo (recomendado) ou aplicado pontualmente (inline). A seguir tens os 3 elementos mais comuns para isso.

1) <style> (CSS interno)

Coloca regras CSS dentro do <head>. É útil para páginas pequenas, protótipos ou exemplos rápidos.

Em projetos reais, normalmente preferimos CSS externo para reutilização e manutenção. 

<head>
  <style>
    body { margin: 0; font-family: system-ui, sans-serif; }
    blockquote { color: blue; }
  </style>
</head>

2) <link> (CSS externo)

Liga a página a um ficheiro .css. É a opção mais usada porque permite que várias páginas partilhem o mesmo estilo. 

<head>
  <link rel="stylesheet" href="estilos/cursos.css">
</head>

Atributos úteis hoje: rel, href, media (ex.: impressão) e, quando aplicável, integrity / crossorigin (CDNs).

3) <span> (texto “inline”)

O <span> não cria um “bloco” novo: serve para aplicar estilo a uma parte pequena de texto dentro de um parágrafo. 

<p>
  Texto normal e um
  <span style="color:red; font-weight:700">destaque</span>.
</p>

<p>
  Usando class: <span class="verm">vermelho</span>
</p>

<p>
  Usando id: <span id="italic">itálico</span>
</p>

Texto normal e um destaque.

Usando class: vermelho

Usando id: itálico

Dica: usa class para estilos reutilizáveis e id para algo único na página.

Boas práticas (hoje)

  • Prefere CSS externo para manter o projeto organizado.
  • Evita estilos inline em excesso (dificulta manutenção).
  • Usa nomes de classes claros (ex.: .btn, .card, .nav).
  • Usa HTML semântico (<header>, <main>, <footer>).
