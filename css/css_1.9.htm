Capítulo 2 — CSS no HTML

Tipos de seletores

1) Seletor simples (tipo)

Aplica estilo a todas as tags de um tipo (ex.: todos os p).

p {
  color: green;
}

2) Seletor contextual (descendente)

Aplica estilo apenas quando um elemento está dentro de outro (ex.: b dentro de p).

p b {
  color: blue;
}

3) Seletor por classe

Aplica estilo a qualquer elemento com class="exemplo". Reutilizável em vários elementos.

.exemplo {
  color: red;
}

4) Seletor por ID

Aplica estilo a um elemento específico com id="chumbo". Deve ser único na página.

#chumbo {
  color: gray;
}

Dica atual: em projetos modernos usa-se muito classes (mais flexíveis) e evita-se abusar de IDs para styling.
