Comentários em CSS

Organização · Legibilidade

Comentários em CSS servem para documentar o código: explicar decisões, separar secções (ex.: “botões”, “layout”, “cores”), deixar notas e facilitar manutenção. Eles não afetam o estilo final.

Em CSS, comentários começam com /* e terminam com */. 

/* Links do site:
   - Vermelhos
   - Sem sublinhado
   - Sublinhado apenas ao passar o rato
*/
a:link,
a:visited {
  color: red;
  text-decoration: none;
}

a:hover {
  text-decoration: underline;
}

/* Acessibilidade: destaque quando navegas com teclado */
a:focus-visible {
  outline: 3px solid rgba(59,130,246,.55);
  outline-offset: 2px;
}
