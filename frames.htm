O que são frames?

Os frames eram uma extensão antiga do HTML que permitia dividir a janela do browser em várias áreas independentes. Cada área (frame) carregava um documento HTML diferente.

Essa divisão podia ser:

  • Vertical – usando o atributo COLS
  • Horizontal – usando o atributo ROWS
⚠️ Importante: O uso de <frameset> e <frame> está obsoleto em HTML5 e não deve ser usado em projetos atuais.

Exemplo histórico (frames verticais)

Código antigo

<html>
<frameset cols="25%,75%">
  <frame src="menu.html" name="menu">
  <frame src="conteudo.html" name="conteudo">
</frameset>
</html>

O que fazia

Dividia o ecrã em duas colunas:

  • 25% para o menu
  • 75% para o conteúdo

Ao clicar num link no menu, apenas a frame do conteúdo era atualizada.

Frames horizontais

<frameset rows="20%,80%">
  <frame src="topo.html">
  <frame src="conteudo.html">
</frameset>

Neste caso, a página era dividida horizontalmente: um topo fixo e uma área principal.

Como se faz hoje (forma correta)

Layout com CSS Grid

.layout {
  display: grid;
  grid-template-columns: 240px 1fr;
  min-height: 100vh;
}

Usado para menus laterais e estruturas fixas.

Uso de iframe (quando necessário)

<iframe
  src="pagina.html"
  title="Conteúdo incorporado"
  width="100%"
  height="400"
  loading="lazy">
</iframe>

Usado apenas para incorporar conteúdo externo ou isolado.
