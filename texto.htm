Formatação de texto

Em HTML podemos destacar partes do texto (negrito, itálico, etc.). No entanto, em HTML5, recomenda-se usar tags semânticas (que explicam o significado do texto) e deixar o “visual” para o CSS.

Tags mais usadas

Efeito / significado Tag (recomendada) Quando usar
Ênfase (normalmente itálico) <em>...</em> Para dar ênfase a uma palavra/expressão.
Importância (normalmente negrito) <strong>...</strong> Para indicar algo importante (ex.: aviso, palavra-chave).
Sublinhado (visual) <u>...</u> Evita usar para “decorar” (pode parecer link). Preferir CSS.
Pequena nota <small>...</small> Rodapés, notas, detalhes menos relevantes.
Texto sobrescrito <sup>...</sup> Expoentes: x2, notas (1), etc.
Texto subscrito <sub>...</sub> Fórmulas: H2O, etc.
Código/monoespaçado <code>...</code> Para mostrar comandos, tags, snippets de código.

Nota: <b> e <i> ainda funcionam, mas hoje preferimos <strong> e <em> por serem mais semânticos.

Tags antigas (evitar hoje)

  • <tt> (typewriter) – obsoleta → usa <code> ou CSS.
  • <big> – obsoleta → usa CSS (font-size).
  • <blink> – não é suportada na maioria dos browsers → evita.

Exemplo (HTML + resultado esperado)

Este texto está em <strong>negrito semântico</strong>.<br> Este texto está em <em>itálico semântico</em>.<br> Este texto está <u>sublinhado</u> (usar com cuidado).<br> <p> Fórmula: H<sub>2</sub>O e potência: x<sup>2</sup>. </p> Para mostrar tags, usa <code>&lt;p&gt;</code>.

Podes ver um exemplo (se o teu projeto ainda tiver essa pasta).

Forma moderna: formatar com CSS

Em projetos reais, o mais comum é usar classes e CSS para controlar o aspeto:

<p class="aviso">Isto é um aviso.</p> /* CSS */ .aviso { font-weight: 700; }
