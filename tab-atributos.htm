Em HTML antigo, era comum usar atributos no <table> como align, width, height, border e bgcolor. Hoje, muitos destes são obsoletos e a recomendação é: HTML para estrutura e CSS para aparência.

O que “controla” uma tabela hoje

  • Largura e alinhamento: width, max-width, margin no CSS
  • Bordas: border, border-collapse, border-radius
  • Espaçamento: padding em th/td
  • Cores e destaque: background, “zebra stripes”, hover
  • Semântica: <caption>, <thead>, <tbody>, scope em <th>

Exemplo antigo (não recomendado)

<table align="right" width="50%" border="5" bgcolor="#FF8080">
  ...
</table>
Porquê evitar? Mistura conteúdo com apresentação, dificulta manutenção e não é a forma moderna de construir páginas.

Exemplo moderno (recomendado)

<table class="minha-tabela">
  <caption>Exemplo de tabela</caption>
  <thead>
    <tr>
      <th scope="col">Coluna 1</th>
      <th scope="col">Coluna 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Linha 1</td>
      <td>Linha 1</td>
    </tr>
  </tbody>
</table>

/* CSS */
.minha-tabela{ width: 100%; border-collapse: collapse; }
.minha-tabela th, .minha-tabela td{ padding: 10px 12px; border: 1px solid #ddd; }
