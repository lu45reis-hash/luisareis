Parágrafo e quebra de linha

Em HTML, podes escrever o texto normalmente, mas há uma diferença importante: o browser não interpreta a tecla Enter como “fim de parágrafo”. Para isso, usamos tags específicas.

As tags principais para este tema são: <p> (parágrafo), <br> (quebra de linha) e <hr> (linha horizontal/separador).

1) Parágrafos: <p>

Usa <p> para criar parágrafos. O browser adiciona automaticamente um espaço (margem) antes/depois do parágrafo — por isso não precisas de repetir <p> para “fazer linhas em branco”.

<h1>Exemplo de parágrafos</h1> <p> Este texto está no mesmo parágrafo, mesmo que no código uses Enter. </p> <p> Este já é outro parágrafo. </p> <hr> <p> Depois do <hr> fica uma linha horizontal a separar conteúdos. </p>

Se queres separar secções, usa <hr> ou CSS — não “empilhes” parágrafos vazios.

Vê um exemplo com estes comandos.

2) Quebra de linha: <br>

Em algumas situações, precisas de ir para a linha seguinte sem criar um novo parágrafo (sem espaço extra). Nesses casos, usa <br>.

<h1>Exemplo de quebra de linha</h1> Este texto está na primeira linha.<br> Este texto continua na linha seguinte.<br> E este na próxima, sem criar novo parágrafo.

Repara na diferença entre <p> e <br> neste exemplo.
