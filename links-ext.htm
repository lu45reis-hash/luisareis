Links externos

Uma das grandes vantagens da Web é o hipertexto: a possibilidade de ligar páginas entre si através de hiperligações (links).

Em HTML, os links são criados com a tag <a> e o atributo href.

Sintaxe básica

<a href="https://exemplo.com">Texto do link</a>

O que é um URL?

URL significa Uniform Resource Locator e é o “endereço” de um recurso na Internet (uma página, uma imagem, um ficheiro, etc.).

Exemplo de um URL:

https://www.exemplo.com/pasta/pagina.html
  • https → protocolo (forma de acesso; hoje em dia usa-se quase sempre HTTPS)
  • www.exemplo.com → domínio/servidor
  • /pasta/pagina.html → caminho (path) do recurso

Abrir o link numa nova aba

Para abrir um link noutra aba do navegador, usa target="_blank".
Por segurança, recomenda-se sempre adicionar rel="noopener noreferrer".

<a href="https://github.com" target="_blank" rel="noopener noreferrer"> Descobrir projetos incríveis no GitHub </a>

Resultado:

Descobrir projetos incríveis no GitHub →

Link numa imagem

Também podes usar uma imagem como link: basta colocar a tag <img> dentro do <a>.

<a href="https://unsplash.com" target="_blank" rel="noopener noreferrer"> <img src="imagens/foto-paisagem.jpg" alt="Paisagem incrível do Parque Nacional de Peneda-Gerês" width="300" height="200"> </a>

Resultado:

Dica: quando usas uma imagem como link, é muito importante ter um texto alternativo (alt) descritivo. Controla a aparência (borda, sombra, cantos arredondados...) com CSS.

Enviar e-mail com mailto:

Em vez de abrir um site, podes criar um link que abre diretamente o cliente de e-mail:

<a href="mailto:exemplo@dominio.pt">Enviar e-mail</a>

Podes ainda sugerir assunto e corpo da mensagem:

<a href="mailto:exemplo@dominio.pt?subject=Dúvida%20HTML&body=Olá!%20Tenho%20uma%20dúvida..."> Contactar </a>

Se ainda tiveres os ficheiros antigos, podes ver o exemplo prático.
