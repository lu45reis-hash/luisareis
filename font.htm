Em HTML antigo usava-se a tag <font> para alterar tipo, tamanho e cor do texto. Em HTML5, o <font> é obsoleto. O recomendado é usar CSS.

Antigo (não recomendado)

<font face="Arial" size="7" color="#008000">
  Texto qualquer
</font>

Atual (recomendado)

/* no CSS */
.titulo{
  font-family: Arial, system-ui, sans-serif;
  font-size: 2rem;
  color: #008000;
}

/* no HTML */
<p class="titulo">Texto qualquer</p>

O que controlas com CSS

  • font-family — tipo de letra (ex.: system-ui, Arial, serif…)
  • font-size — tamanho (ex.: 16px, 1rem, 2rem)
  • color — cor do texto
  • font-weight, font-style, text-decoration — negrito, itálico, sublinhado…
Boa prática: evita “pintar” cada palavra com estilos diferentes. Em projetos reais, cria classes (ex.: .titulo, .destaque) e reutiliza. Fica consistente e mais fácil de manter.
