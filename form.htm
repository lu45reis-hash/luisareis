O que são formulários?

Um formulário é o principal meio de interação entre o utilizador e uma página web. É através dele que o utilizador introduz dados (texto, escolhas, ficheiros, etc.) que podem ser enviados para um servidor.

A criação de formulários envolve normalmente duas partes:

  • HTML – definição dos campos e da estrutura visual;
  • Script do servidor – tratamento dos dados recebidos (PHP, Python, JavaScript, etc.).

Estrutura básica de um formulário

Código HTML5

<form action="processar.php" method="post">
  <input type="text" name="nome">
  <input type="email" name="email">
  <button type="submit">Enviar</button>
</form>

O que faz cada parte

  • <form> – delimita o formulário
  • action – destino dos dados
  • method – forma de envio
  • <input> – campos de entrada
  • <button> – envio do formulário

O atributo method

  • POST – envia os dados no corpo da requisição (mais seguro e sem limite prático de tamanho).
  • GET – envia os dados pela URL (limitado e visível no endereço).
- Em formulários modernos, o método POST é o mais utilizado.

Principais tipos de campos

  • <input> – texto, email, password, checkbox, radio, etc.
  • <select> – listas de opções
  • <textarea> – textos longos
- Em HTML5 existem novos tipos como email, number, date e validação automática.
