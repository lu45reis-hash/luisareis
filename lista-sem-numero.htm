O que são listas não numeradas

As listas não numeradas são usadas quando a ordem dos itens não é relevante. Cada item é apresentado com um marcador (normalmente um ponto, círculo ou quadrado).

Em HTML, este tipo de lista é criado com o elemento <ul> (unordered list) e cada item da lista é definido com o elemento <li>.

Sintaxe básica

<ul>
  <li>Primeiro item</li>
  <li>Segundo item</li>
  <li>Terceiro item</li>
</ul>

Resultado no browser:

  • Primeiro item
  • Segundo item
  • Terceiro item

O browser apresenta automaticamente um marcador para cada item da lista.

Listas hierárquicas (listas dentro de listas)

É possível criar listas com vários níveis, inserindo uma lista dentro de outra. Esta técnica é muito utilizada em menus, índices e estruturas de tópicos.

Código HTML

<ul>
  <li>Item principal
    <ul>
      <li>Sub-item 1</li>
      <li>Sub-item 2</li>
    </ul>
  </li>
  <li>Outro item principal</li>
</ul>

Resultado no browser:

  • Item principal
    • Sub-item 1
    • Sub-item 2
  • Outro item principal

Cada nível da lista utiliza automaticamente um tipo de marcador diferente.
