No passado, era comum definir “papel de parede” com o atributo BACKGROUND no <body>. Hoje isso é considerado obsoleto. A forma correta (e recomendada) é usar CSS.

Antigo (não recomendado)

<body background="imagem.jpg">

Atual (recomendado)

body{
  background-color: #f6f7fb;                /* cor de fundo */
  background-image: url("imagens/fundo.jpg"); /* imagem de fundo */
  background-repeat: repeat;               /* repetir (padrão) */
  /* alternativas comuns:
     background-repeat: no-repeat;
     background-size: cover;
     background-position: center;
  */
}

Boas práticas (hoje)

  • Escolhe fundos que não prejudiquem a leitura (contraste é essencial).
  • Evita imagens muito “ruidosas” atrás de texto. Se precisares, usa um overlay (camada suave).
  • Para páginas responsivas, usa muitas vezes background-size: cover.
Dica: em sites modernos, é comum usar fundos discretos e conteúdo em “cards” brancos para garantir legibilidade e um aspeto profissional.
