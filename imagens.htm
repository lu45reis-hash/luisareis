Imagens

Inserir uma imagem numa página Web é simples: basta ter o ficheiro de imagem e usar a tag <img>. O browser vai carregá-la a partir do caminho indicado no atributo src.

Formatos recomendados hoje

  • JPG/JPEG: fotografias (boa compressão).
  • PNG: imagens com transparência (ex.: logótipos, ícones).
  • SVG: gráficos vetoriais (ótimo para logótipos e ícones, sem perder qualidade).
  • WebP: moderno e eficiente (fotografias e imagens gerais).

Evita BMP: é pesado e não é apropriado para a Web.

Sintaxe básica

<img src="imagens/foto.jpg" alt="Descrição da imagem">

O atributo alt é muito importante: é lido por leitores de ecrã e aparece se a imagem não carregar.

Boas práticas (recomendadas)

Boa prática Exemplo Porquê
Usar alt <img ... alt="Logótipo"> Acessibilidade e fallback.
Definir width e height width="600" height="400" Evita “saltos” no layout ao carregar.
Carregamento lazy loading="lazy" Melhora performance (especialmente com muitas imagens).

Exemplo completo (recomendado)

<img src="imagens/duke.png" alt="Mascote Duke" width="160" height="160" loading="lazy">

Alinhamento: hoje faz-se com CSS (não com ALIGN)

Antigamente usava-se ALIGN="left/right", mas hoje isso é feito com CSS. Por exemplo, para pôr uma imagem à esquerda e o texto a contornar:

<img class="img-left" src="imagens/duke.png" alt="Mascote Duke" width="160" height="160"> <p>Texto que contorna a imagem...</p> /* CSS */ .img-left { float: left; margin: 0 12px 12px 0; }

E para alinhar à direita:

<img class="img-right" src="imagens/duke.png" alt="Mascote Duke" width="160" height="160"> <p>Texto que contorna a imagem...</p> /* CSS */ .img-right { float: right; margin: 0 0 12px 12px; }

Caminhos (onde o browser “vai buscar” a imagem)

Se a imagem estiver na mesma pasta do ficheiro HTML:

<img src="stego.jpg" alt="Dinossauro Stego">

Se estiver numa subpasta (por exemplo, imagens/):

<img src="imagens/stego.jpg" alt="Dinossauro Stego">

Se a imagem estiver noutro site (URL completa):

<img src="https://exemplo.com/imagens/icone.png" alt="Ícone">

Atenção ao tamanho e velocidade

Imagens muito pesadas tornam a página lenta. Sempre que possível, otimiza as imagens (reduz dimensões e comprime) antes de as colocar no site.
