Uma página da Web é composta basicamente por textos e comandos especiais (tags) de uma linguagem de marcação chamada HTML, abreviação de HyperText Markup Language.

Esta linguagem é simples e tem como finalidade principal estruturar e formatar o conteúdo apresentado, bem como criar ligações entre páginas através do conceito de hipertexto.

Tal como em qualquer linguagem, o programador deve escrever o código-fonte seguindo regras de sintaxe. Este código é depois interpretado pelo browser, que apresenta o resultado ao utilizador.

O código HTML pode ser escrito em qualquer editor de texto simples. No entanto, a forma mais prática de criar páginas Web é utilizar um editor de código apropriado.

Existem vários editores de HTML disponíveis; contudo, nada substitui o conhecimento da linguagem HTML e das boas práticas de construção de páginas Web.

Editores HTML (atuais)

  • Visual Studio Code (Microsoft) – o mais utilizado atualmente
  • Sublime Text – leve e rápido
  • Notepad++ – simples, ideal para iniciantes
  • Brackets – focado em HTML e CSS
  • WebStorm – editor profissional (pago)

Nota: deves utilizar o editor que melhor se adapta às tuas necessidades e ao teu nível.

O que são TAGS

Quando um browser exibe uma página Web, ele analisa o código HTML e procura por elementos especiais chamados tags, que indicam como o conteúdo deve ser apresentado.

Por exemplo, uma tag pode definir se um texto aparece em negrito ou itálico, o tipo e o tamanho da fonte, ou se um determinado texto é um link para outra página.

As tags normalmente funcionam em pares (abertura e fecho), delimitando o conteúdo afetado. No entanto, existem também tags que não necessitam de fecho.

Estrutura básica de um elemento HTML:

<elemento> Conteúdo </elemento>

O rótulo de fecho possui o mesmo nome do rótulo inicial, precedido por uma barra (/). Alguns elementos podem ainda conter atributos.

Sequências de escape (entidades)

Para apresentar certos caracteres reservados em HTML, utilizam-se entidades especiais:

Caractere Entidade
<&lt;
>&gt;
&&amp;
"&quot;

Em HTML moderno, recomenda-se escrever as tags em letras minúsculas.
