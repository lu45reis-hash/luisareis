Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem usada para estruturar conteúdo na Web. Neste módulo vais perceber a lógica dos “elementos” (tags), regras básicas e a estrutura mínima de um documento.

O que é HTML?

O HTML é uma linguagem de marcação (não é linguagem de programação) composta por texto e elementos que descrevem a estrutura da página: títulos, parágrafos, listas, ligações, imagens, etc. O browser interpreta esse “marcado” e apresenta o resultado ao utilizador.

Nota atual (boas práticas): hoje usa-se UTF-8 e HTML5 com elementos semânticos (<header>, <main>, <footer>…).

Regras básicas

  • O browser normalmente ignora espaços extra e quebras de linha no HTML (exceto em <pre> ou com CSS).
  • Muitos elementos têm abertura e fecho: <tag>...</tag>. (Alguns são “vazios”, como <img>.)
  • Em HTML, maiúsculas/minúsculas não costuma ser problema, mas é recomendado escrever de forma consistente. Em ficheiros e links, porém, o nome do ficheiro conta em muitos servidores (Linux).
  • Em HTML5 com UTF-8, podes escrever acentos diretamente (ex.: “introdução”). Entidades como &uacute; continuam válidas quando precisares.

Estrutura básica de um documento HTML

Exemplo mínimo (HTML5): 

<!doctype html>
<html lang="pt-PT">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Curso de HTML</title>
  </head>
  <body>
    Conteúdo do documento
  </body>
</html>

O que significa cada parte:

  • <!doctype html>: indica HTML5 ao browser.
  • <html>: raiz do documento; usa lang para acessibilidade e SEO.
  • <head>: metadados (charset, título, estilos, scripts).
  • <title>: título mostrado no separador do browser.
  • <body>: conteúdo visível da página.
