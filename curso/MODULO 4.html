Parágrafos e quebras de linha

Aprende a separar ideias em parágrafos (<p>), a alinhar texto (hoje com CSS) e a forçar quebras de linha (<br>). Estes elementos são básicos para tornar um texto legível numa página Web.

Descritores de parágrafos

Um parágrafo é marcado com <p>…</p>. Em HTML moderno, é recomendado fechar sempre o parágrafo para evitar ambiguidades.

Texto formatado (resultado)

Primeira linha. Segunda linha.

Primeira linha.

Segunda linha.

HTML

Primeira linha. Segunda linha.

Primeira linha.
<p>Segunda linha.</p>

Nota: para “espaço” entre parágrafos, os browsers aplicam margens por defeito (CSS). Mais tarde podes personalizar isso.

Alinhamento de texto

Historicamente usava-se align="left|center|right" em elementos. Em HTML5 isso é considerado legado; a forma moderna é usar CSS com text-align.

Texto formatado (resultado)

ESQUERDA

CENTRO

DIREITA

HTML (moderno)

<h3 style="text-align:left;">ESQUERDA</h3>
<h3 style="text-align:center;">CENTRO</h3>
<h3 style="text-align:right;">DIREITA</h3>

Boa prática: em projetos reais, evita style="" repetido e usa classes CSS (ex.: .center { text-align:center; }).

Quebras de linha

A quebra de linha é feita com <br>. Não tem fecho. Usa-se para quebras “dentro da mesma ideia” (ex.: moradas, poemas). Para separar ideias, prefere <p>.

Texto formatado (resultado)

Primeira linha. Segunda linha.

Primeira linha.
Segunda linha.

HTML

Primeira linha. Segunda linha.

Primeira linha.<br>
Segunda linha.

Exercício prático

Escreve/edita o HTML e clica em Processar para veres o resultado no preview. Experimenta trocar <p> por <br> e observar a diferença.

Dica: usa <div style="text-align:right;">…</div> para alinhar um bloco todo à direita.
