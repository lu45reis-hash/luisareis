📘 Módulo 6

Marcadores e listas numeradas

Listas ajudam a organizar informação: <ul> (desordenadas), <ol> (ordenadas) e <dl> (definições). Neste módulo vais aprender a criar listas simples e aninhadas.

Listas desordenadas (marcadores) — <ul>

Uma lista desordenada usa <ul> e cada item é um <li>. O atributo type é legado (funciona, mas hoje controla-se com CSS).

Texto formatado (resultado)

LISTA DESORDENADA

  • ITEM 1
  • ITEM 2
  • ITEM 3

HTML + CSS (moderno)

<ul style="list-style-type: circle;">
  <li>ITEM 1</li>
</ul>

<ul style="list-style-type: square;">
  <li>ITEM 2</li>
</ul>

<ul style="list-style-type: disc;">
  <li>ITEM 3</li>
</ul>

(Legado) também existe: <ul type="circle">, <ul type="square">, <ul type="disc">.

Boa prática para sub-listas: coloca a lista “filha” dentro do <li> correspondente (fica mais correto e previsível).

Texto formatado (resultado)

LISTA DESORDENADA (COM SUB-ITENS)

  • ITEM 1
    • SUB-ITEM 1
  • ITEM 2
    • SUB-ITEM 2

HTML (aninhado corretamente)

<ul>
  <li>ITEM 1
    <ul style="list-style-type: circle;">
      <li>SUB-ITEM 1</li>
    </ul>
  </li>

  <li>ITEM 2
    <ul style="list-style-type: circle;">
      <li>SUB-ITEM 2</li>
    </ul>
  </li>
</ul>

Listas ordenadas (numeradas) — <ol>

Uma lista ordenada usa <ol>. Podes mudar o tipo de numeração com type e definir o ponto de partida com start.

Texto formatado (resultado)

LISTA ORDENADA NUMÉRICA

  1. Primeiro item
  2. Segundo item

LISTA ORDENADA ALFABÉTICA (A)

  1. Primeiro item
  2. Segundo item

LISTA ORDENADA ROMANA (I)

  1. Primeiro item
  2. Segundo item

COM VALOR INICIAL

  1. Terceiro item
  2. Quarto item
  3. Quinto item

HTML

<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
</ol>

<ol type="A">
  <li>Primeiro item</li>
  <li>Segundo item</li>
</ol>

<ol type="I">
  <li>Primeiro item</li>
  <li>Segundo item</li>
</ol>

<ol start="3">
  <li>Terceiro item</li>
  <li>Quarto item</li>
  <li>Quinto item</li>
</ol>

Dica moderna: para começar num número e manter semântica, start é perfeitamente aceitável. Para estilos mais avançados, usa CSS.

Listas de glossário (definições) — <dl>

Estas listas são úteis para termos e respetivas definições. Usa-se <dl>, com <dt> (termo) e <dd> (definição).

Texto formatado (resultado)

LISTA DE GLOSSÁRIO

HTML
HyperText Markup Language
WWW
World Wide Web

HTML

<dl>
  <dt><strong>HTML</strong></dt>
  <dd>HyperText Markup Language</dd>

  <dt><strong>WWW</strong></dt>
  <dd>World Wide Web</dd>
</dl>

Exercício prático

Edita o HTML e clica em Processar para veres o resultado no preview. Experimenta criar uma lista com sub-itens e uma mini lista de glossário.

Dica: tenta mudar o estilo da lista com list-style-type (CSS) e compara com type="".
