Hiperlinks

Hiperlinks permitem navegar entre páginas e também dentro da mesma página. O elemento principal é <a> com o atributo href.

Navegação dentro do mesmo documento (âncoras)

Em HTML moderno, marca-se o destino com um id (em vez de name). O link aponta para #id.

Exemplo (resultado)

Ir para o local de destino

(Conteúdo intermédio…) Faz scroll e clica no link acima.

Ponto de destino (id="destino-demo")

HTML

<!-- ponto de origem -->
<a href="#marca">Ir para o local de destino</a>

<!-- ponto de destino -->
<div id="marca">Ponto de destino</div>

Nota: o antigo <a name="marca"> ainda aparece em material antigo, mas hoje prefere-se id.

Navegação entre documentos (relativo, absoluto e URL)

O atributo href aceita caminhos relativos (ficheiros na mesma pasta), caminhos absolutos no site (começam com /) e URLs completas (http/https).

Exemplos (resultado)

  • Link relativo para o índice do curso
  • Link para âncora noutro ficheiro (relativo)
  • Link externo (abre noutra aba)
  • Link para email (mailto)
  • Link para telefone (tel)

(*No teu site, troca “example.com” por um URL real.)

HTML

<a href="curso.HTM">Ver índice do Curso</a>

<a href="intro.htm#marca">Ir para a âncora marca</a>

<a href="https://example.com"
   target="_blank"
   rel="noopener noreferrer">
  Abrir site externo
</a>

<a href="mailto:luisa.reis@aebatalha.ccems.pt">Enviar email</a>
<a href="tel:+351000000000">Telefonar</a>

Boas práticas rápidas

  • Links externos: se usares target="_blank", adiciona rel="noopener noreferrer".
  • Texto do link: evita “clique aqui”. Prefere algo descritivo (ex.: “Ver índice do curso”).
  • Âncoras: usa id únicos e fáceis de ler (ex.: id="contactos").
  • Segurança: evita links para caminhos locais do Windows (c:\...) — no browser isso não funciona como numa página web publicada.

Exercício prático

Edita o HTML e clica em Processar para veres o resultado no preview. Experimenta criar: (1) um link interno para uma secção, (2) um link para o índice, e (3) um link externo.

Dica: cria uma secção com id="fim" e um link <a href="#fim">...</a>.
