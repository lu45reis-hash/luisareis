Barras horizontais

A tag <hr> cria uma separação visual entre secções. Antigamente usava-se width e size no próprio HTML; hoje é preferível controlar largura/espessura com CSS.

O que é <hr>?

A barra horizontal é uma forma rápida de indicar uma mudança de tema ou separação de conteúdo. Em HTML5, <hr> tem um significado semântico: “mudança de tópico” dentro do mesmo contexto.

Dica: usa com moderação. Muitas vezes uma boa hierarquia de títulos e espaçamento (CSS) faz o mesmo trabalho.

Nota: width e size em <hr> são antigos; funciona em muitos browsers, mas o ideal é usar CSS.

Versão moderna (com CSS)

O mais recomendado é criar o <hr> e depois definir estilo via CSS:

CSS

hr{
  border: 0;
  height: 2px;
  background: #ccc;
  margin: 16px 0;
}

hr.curto{ width: 40%; }
hr.grosso{ height: 8px; border-radius: 999px; }

HTML

<hr>
<hr class="curto">
<hr class="grosso">
