Tabela HTML — código, resultado e explicação

Código HTML

<table>
  <tr>
    <th>Linha 1 · Coluna 1</th>
    <th>Linha 1 · Coluna 2</th>
  </tr>
  <tr>
    <td>Linha 2 · Coluna 1</td>
    <td>Linha 2 · Coluna 2</td>
  </tr>
</table>

Resultado

Linha 1 · Coluna 1 Linha 1 · Coluna 2
Linha 2 · Coluna 1 Linha 2 · Coluna 2
Exemplo simples de uma tabela HTML

Explicação das tags principais

  • <table> — cria a tabela. Todo o conteúdo tabular deve estar dentro desta tag.
  • <tr> (Table Row) — define uma linha da tabela.
  • <th> (Table Header) — define uma célula de cabeçalho. O texto aparece, por defeito, a negrito e centrado.
  • <td> (Table Data) — define uma célula de dados “normal”.
  • <caption> — legenda da tabela (opcional), usada para descrever o conteúdo da tabela.
