Tabela HTML — código, resultado e explicação

Código antigo (não recomendado)

<table align="right"
       width="50%"
       height="50%"
       border="5"
       bgcolor="#FF8080">

  <tr>
    <th>Linha 1 Coluna 1</th>
    <th>Linha 1 Coluna 2</th>
  </tr>

  <tr>
    <td>Linha 2 Coluna 1</td>
    <td>Linha 2 Coluna 2</td>
  </tr>
</table>

Resultado (forma moderna)

Linha 1 · Coluna 1 Linha 1 · Coluna 2
Linha 2 · Coluna 1 Linha 2 · Coluna 2
Exemplo de tabela formatada com CSS

Explicação

  • O elemento <table> cria a tabela e deve conter apenas dados tabulares.
  • A tag <tr> define uma linha da tabela.
  • <th> define uma célula de cabeçalho (texto em destaque).
  • <td> define uma célula de dados.
  • Atributos como align, bgcolor, border e height são hoje considerados obsoletos.
  • Em HTML5, o alinhamento, cores, tamanhos e bordas devem ser definidos com CSS.
Nota importante: Tabelas devem ser usadas apenas para dados. O layout das páginas deve ser feito com CSS (Flexbox ou Grid).
