Exemplo de Hiperligações

O conceito de hipertexto permite criar documentos que fazem referência a outros através de hiperligações (links). Caso queira aprender mais sobre HTML, visite o site W3Schools – HTML .

Ligação para envio de e-mail

Em vez de redirecionar o utilizador para outra página Web, pode-se criar uma ligação para envio de e-mail, utilizando o esquema mailto:.

Exemplo: Enviar e-mail

A sintaxe geral é: 

<a href="mailto:nome@endereco">Enviar e-mail</a>
