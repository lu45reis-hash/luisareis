Exemplos de links locais (no mesmo servidor)

Os links locais são utilizados para ligar páginas que se encontram no mesmo site ou servidor. Para isso, basta indicar o caminho relativo do ficheiro que queremos aceder.

  • Voltar para o índice geral
  • Voltar para a lição «Links internos»

Sintaxe utilizada

<a href="../index.html">Voltar para o índice geral</a>

<a href="../links-int.htm">Voltar para a lição «Links internos»</a>
