Entrada de dados em formulários

Em HTML existem três tipos principais de campos para a entrada de dados:

  • <input>
  • <select>
  • <textarea>

Todos os campos possuem o atributo name, que identifica o dado enviado ao servidor.

Campo de texto (type="text")

Código

<label>Nome
  <input type="text" name="nome">
</label>

Resultado

É o campo mais comum. Permite a introdução de texto simples.

Campo de palavra-passe (type="password")

<label>Senha
  <input type="password" name="senha">
</label>

O texto introduzido é ocultado. Ideal para palavras-passe.

Caixas de seleção (checkbox)

<label>
  <input type="checkbox" name="browser" value="Chrome">
  Chrome
</label>

Permitem selecionar várias opções.

Botões de opção (radio)

<label>
  <input type="radio" name="browser">
  Chrome
</label>

Apenas uma opção pode ser escolhida.

Botões de ação

<button type="submit">Enviar</button>
<button type="reset">Limpar</button>

Submit envia os dados do formulário. Reset limpa todos os campos.

- Em HTML5, recomenda-se usar <button> em vez de <input type="submit">.
