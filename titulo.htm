Cabeçalhos

Os cabeçalhos (headings) são títulos e subtítulos que organizam o conteúdo de uma página. Em HTML existem seis níveis: <h1> até <h6>.

Eles servem para indicar o início de uma secção ou tópico e ajudam tanto os utilizadores como os motores de pesquisa e leitores de ecrã a compreender a estrutura do documento.

Sintaxe

<h1>Título principal</h1>

Onde o número vai de 1 a 6. O <h1> é o mais importante (normalmente o título principal) e o <h6> é o menos importante.

Exemplo completo

<h1>Cabeçalho 1</h1> <h2>Cabeçalho 2</h2> <h3>Cabeçalho 3</h3> <h4>Cabeçalho 4</h4> <h5>Cabeçalho 5</h5> <h6>Cabeçalho 6</h6>

Como fica na página

Cabeçalho 1

Cabeçalho 2

Cabeçalho 3

Cabeçalho 4

Cabeçalho 5
Cabeçalho 6

Dica: usa cabeçalhos para estrutura, não apenas para “aumentar o tamanho” do texto. Para estilos (tamanho/cor), usa CSS.

Boas práticas (muito importante)

  • Usa um único H1 por página (normalmente o título principal).
  • Não saltes níveis: evita passar de h2 para h4 sem um h3.
  • Os headings devem fazer sentido como um índice do conteúdo.
  • Evita usar headings só para “dar estilo” (isso é trabalho do CSS).
