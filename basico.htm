Estrutura básica de uma página HTML

Tal como noutras linguagens, existem estruturas básicas que organizam um documento HTML. Em HTML5, a estrutura típica inclui <html>, <head> e <body>.

Uma página HTML tem normalmente: um elemento raiz <html>, a secção <head> (com metadados como <title>) e o <body> (com o conteúdo visível).

O <title> define o nome que aparece na aba do browser. O conteúdo principal do site (títulos, texto, imagens, links, tabelas, etc.) fica dentro do <body>.

Uma página Web é composta por títulos, texto, imagens e links — sendo os links os responsáveis por criar ligações entre páginas.

Exemplo (HTML5)

<!doctype html> <html lang="pt-PT"> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, initial-scale=1"> <title>Título da página</title> </head> <body> <!-- Conteúdo da página --> </body> </html>

Usa sempre HTML5 e guarda os ficheiros em UTF-8 para evitar problemas de acentuação.
