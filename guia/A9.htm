Quadros (Frames)

html
  |__ head
  |__ frameset
          |__ frame
          |__ noframes
⚠️ Nota importante: A tecnologia de quadros (frames) foi descontinuada e é considerada obsoleta em HTML5. Hoje em dia recomenda-se fortemente o uso de CSS Flexbox/Grid ou iframe quando necessário. Esta página é mantida apenas para referência histórica.

<frameset> ... </frameset>

Define a estrutura de divisão da janela do navegador em vários quadros. Substitui o elemento <body>.

<frameset cols="25%, 75%"> <frame src="menu.html"> <frame src="conteudo.html"> </frameset>

Atributos principais:

  • cols – divide a página em colunas (larguras)
  • rows – divide a página em linhas (alturas)
  • frameborder – define se as bordas são visíveis (0 = sem bordas)

<frame>

Define cada quadro individual dentro de um <frameset>.

<frame src="pagina.html" name="conteudo" scrolling="auto" noresize>

Atributos mais usados:

  • src – URL do documento a carregar no quadro
  • name – nome do quadro (para usar com o atributo target dos links)
  • scrolling – yes, no ou auto
  • noresize – impede que o utilizador redimensione o quadro

<noframes> ... </noframes>

Conteúdo alternativo que será exibido por navegadores que não suportam frames.

<noframes> <body> <p>O seu navegador não suporta frames.</p> <a href="conteudo.html">Ir para a versão sem frames</a> </body> </noframes>
