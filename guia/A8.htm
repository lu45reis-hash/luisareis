<a> ... </a>

O elemento mais importante do HTML. Cria hiperligações (links) para outras páginas, secções ou recursos.

<a href="https://www.exemplo.pt">Visitar o site</a> <a href="#secção1">Ir para a secção 1</a> <a href="documento.pdf" target="_blank" rel="noopener">Abrir PDF em nova aba</a>

<img>

Insere uma imagem na página.

<img src="foto.jpg" alt="Descrição detalhada da imagem para acessibilidade" width="800" height="600" loading="lazy">

Atributos obrigatórios/recomendados:

  • src — caminho ou URL da imagem (obrigatório)
  • alt — texto alternativo (essencial para acessibilidade e SEO)
  • width e height — dimensões (evita saltos de layout)
  • loading="lazy" — carregamento diferido (melhor performance)

<br>

Quebra de linha simples. Não possui etiqueta de fecho.

Imagens mapeadas (Image Maps)

Permitem criar áreas clicáveis sobre uma imagem.

<img src="mapa.jpg" alt="Mapa interativo" usemap="#meumapa"> <map name="meumapa"> <area shape="rect" coords="10,10,200,150" href="pagina1.html" alt="Região Norte"> <area shape="circle" coords="300,100,60" href="pagina2.html" alt="Região Central"> </map>
Elementos obsoletos — não usar:
• <font>
• <applet> e <param>
• Atributos de apresentação como align, border, hspace, vspace (substituídos por CSS)

Recomendação atual (2026):

  • Sempre use o atributo alt descritivo nas imagens
  • Prefira SVG ou ícones CSS quando possível
  • Use loading="lazy" para melhorar a performance
  • Evite image maps sempre que possível — são pouco acessíveis e difíceis de manter
