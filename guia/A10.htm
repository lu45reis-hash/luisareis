Folhas de Estilo (CSS)

As Folhas de Estilo em Cascata (CSS) permitem separar o conteúdo (HTML) da apresentação visual. É a forma moderna e recomendada de controlar cores, tipografia, layout, espaçamentos, animações e responsividade das páginas web.

Hoje em dia utilizamos:
• CSS3 e módulos mais recentes (Flexbox, Grid, Variables, Container Queries, etc.)
• Folhas de estilo externas (.css)
• Metodologias como BEM, SMACSS ou utilitários (Tailwind)

Formas de incluir CSS

<!-- 1. Folha de estilos externa (recomendada) --> <link rel="stylesheet" href="estilos.css"> <!-- 2. Dentro do <head> --> <style> body { font-family: system-ui, sans-serif; background-color: #f8f9fa; } </style> <!-- 3. Atributo style (usar com moderação) --> <p style="color: #e74c3c; font-weight: 600;">Texto destacado</p>

Exemplos práticos modernos

/* Variáveis CSS (Custom Properties) */ :root { --cor-principal: #007bff; --fonte-principal: system-ui, sans-serif; --espacamento: 1.6rem; } body { font-family: var(--fonte-principal); background-color: #f8f9fa; color: #222; line-height: 1.7; } h1, h2 { color: var(--cor-principal); } a { color: var(--cor-principal); text-decoration: none; } a:hover { text-decoration: underline; color: #0056b3; } /* Layout com Flexbox e Grid */ .container { display: grid; grid-template-columns: repeat(auto-fit, minmax(280px, 1fr)); gap: 2rem; }

Elemento <span>

Usado para aplicar estilo a uma parte específica do texto sem alterar a estrutura semântica.

Este é um <span style="color: #e74c3c; font-weight: 700; background-color: #fff3cd;">trecho importante</span> do texto.

Atributo style

Útil para estilos pontuais, mas deve ser usado com moderação para manter o código limpo e fácil de manter.

Boas práticas atuais

  • Prefira sempre folhas de estilo externas
  • Use variáveis CSS (:root) para cores e tamanhos repetidos
  • Utilize Flexbox e CSS Grid para layouts
  • Escreva CSS responsivo com media queries ou container queries
  • Mantenha a separação clara entre conteúdo (HTML) e apresentação (CSS)

Recomendação final: Aprenda CSS moderno (Flexbox + Grid + Variáveis) em vez de depender de atributos antigos como align, bgcolor ou font.
