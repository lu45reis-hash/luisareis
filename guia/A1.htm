<!DOCTYPE html>

Este descritor não faz parte do HTML propriamente dito. Deve aparecer apenas uma vez, no início do documento, para indicar a versão do HTML utilizada.

A maioria dos navegadores não reclama da sua ausência, mas o seu uso é fortemente recomendado, pois facilita a validação e garante uma renderização mais correcta.

<html> ... </html>

É o elemento raiz do documento HTML. Deve envolver todo o conteúdo da página.

Deve conter: <head> e <body> (ou <head>, <frameset> e <noframes> no caso de quadros).

<html lang="pt-PT"> <head>...</head> <body>...</body> </html>

<head> ... </head>

Contém informações sobre o documento (metadados) que não são exibidas diretamente na página.

Pode conter: <title>, <meta>, <link>, <style>, <script>, etc.

<body> ... </body>

Contém todo o conteúdo visível da página (texto, imagens, tabelas, formulários, etc.).

Atributos principais (atualmente recomendados usar CSS):

  • background – imagem de fundo
  • bgcolor – cor de fundo
  • text, link, vlink, alink – cores do texto e links
<body style="background-color: #f8f9fa; color: #222;">

Comentários

Os comentários não são exibidos pelo navegador e servem para documentar o código ou temporariamente desativar partes do HTML.
