Nota: Os nomes das cores a seguir são reconhecidos pelo Netscape Navigator e pela maioria dos navegadores modernos (baseados no sistema X-Window). Qualquer navegador pode interpretar diretamente o código hexadecimal (#rrggbb).

Dica atual: Em CSS moderno recomenda-se usar sempre os códigos hexadecimais (#rrggbb) ou as funções rgb() / hsl() para maior precisão e compatibilidade.
