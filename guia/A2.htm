<title> ... </title>

Define o título da página, que aparece na aba do navegador e é muito importante para motores de busca (SEO) e indexação.

Deve conter: apenas texto simples (não pode conter outras etiquetas HTML).

<title>Guia de Referência HTML: Elementos do Cabeçalho</title>

<meta>

Utilizado para fornecer metadados sobre o documento (informações para navegadores, motores de busca e servidores HTTP).

Atributos principais:

  • name + content — metadados gerais (ex: descrição, autor, palavras-chave)
  • http-equiv + content — simula cabeçalhos HTTP (ex: charset, refresh, etc.)
<meta charset="UTF-8"> <meta name="description" content="Guia completo de referência HTML"> <meta name="viewport" content="width=device-width, initial-scale=1.0">

<link>

Estabelece relações entre o documento atual e outros recursos (folhas de estilo, ícones, feeds, etc.).

É o elemento mais usado para ligar uma folha de estilos externa:

<link rel="stylesheet" href="estilos.css">

<base>

Define a URL base para todos os links relativos da página. Também pode definir o destino padrão dos links.

<base href="https://www.exemplo.pt/"> <base target="_blank">

<isindex>

Elemento obsoleto (deprecated). Criava uma caixa de pesquisa simples no topo da página. Hoje em dia deve ser substituído por formulários HTML modernos.

<style> ... </style>

Permite inserir regras de CSS diretamente na página.

<style> body { font-family: Arial, sans-serif; } h1 { color: #007bff; } </style>

<script> ... </script>

Utilizado para incluir código JavaScript (ou outras linguagens de scripting) na página.

<script src="script.js"></script>
