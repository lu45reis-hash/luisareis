Formulários

form
  |__ input
  |__ select
  |      |__ option
  |__ textarea

<form> ... </form>

Define um formulário interativo. Todos os campos de entrada devem estar dentro dele.

<form action="processar.php" method="POST"> ... campos do formulário ... </form>

Atributos principais:

  • action – URL onde os dados serão enviados (obrigatório)
  • method – GET ou POST (recomenda-se POST para a maioria dos casos)
  • enctype – forma de codificação dos dados (útil para upload de ficheiros)

<input>

Elemento mais versátil dos formulários. O seu comportamento muda conforme o atributo type.

Tipos mais comuns de <input>

type="text" e type="password"

<input type="text" name="nome" size="30" maxlength="100"> <input type="password" name="senha" size="20">

type="radio" (botões de rádio)

<input type="radio" name="genero" value="masculino" checked> Masculino <input type="radio" name="genero" value="feminino"> Feminino

type="checkbox" (caixas de seleção)

<input type="checkbox" name="interesses" value="html"> HTML <input type="checkbox" name="interesses" value="css"> CSS

type="submit" e type="reset"

<input type="submit" value="Enviar formulário"> <input type="reset" value="Limpar campos">

type="hidden"

<input type="hidden" name="userid" value="12345">

<select> ... </select>

Cria um menu de opções (caixa de seleção).

<select name="pais"> <option value="pt">Portugal</option> <option value="br" selected>Brasil</option> <option value="ao">Angola</option> </select>

<option> ... </option>

Define cada opção dentro de um <select>. O atributo selected marca a opção pré-selecionada.

<textarea> ... </textarea>

Campo de texto multilinha (área de texto).

<textarea name="mensagem" rows="8" cols="50"> Escreva aqui a sua mensagem... </textarea>

Nota atual: Muitos atributos antigos de apresentação foram substituídos por CSS ou por novos tipos de <input> (email, url, number, date, etc.).
