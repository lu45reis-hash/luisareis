Cabeçalhos, Blocos e Separadores

body

  |__ p

  |__ pre

  |__ center

<h1> a <h6>

Os cabeçalhos definem a hierarquia de títulos da página. Existem seis níveis, do mais importante (<h1>) ao menos importante (<h6>).

Os navegadores exibem os cabeçalhos com tamanhos decrescentes e geralmente em negrito.

<h1>Título Principal</h1> <h2>Subtítulo</h2> <h3>Secção</h3>

<p> ... </p>

Define um parágrafo de texto. O fecho </p> é opcional em HTML5, mas recomendado para maior clareza.

<p>Este é um parágrafo normal de texto.</p>

<hr>

Cria uma linha horizontal de separação (regra horizontal).

Em HTML5 os atributos de apresentação (size, width, align, noshade) foram descontinuados. Deve-se usar CSS.

<hr style="border: 2px solid #007bff; margin: 30px 0;">

<pre> ... </pre>

Texto pré-formatado. Preserva espaços, tabulações e quebras de linha exatamente como estão no código. É exibido numa fonte mono-espaçada (geralmente Courier).

<pre> function ola() { console.log("Olá Mundo!"); } </pre>

<blockquote> ... </blockquote>

Usado para citar um bloco de texto (geralmente indentado à direita e à esquerda). Ideal para citações longas.

<blockquote> <p>O importante não é vencer todos os dias, mas lutar sempre.</p> <cite>— Desconhecido</cite> </blockquote>

<address> ... </address>

Define informações de contacto ou assinatura do autor. Normalmente é exibido em itálico.

<address> Escrito por João Silva
<a href="mailto:joao@exemplo.pt">joao@exemplo.pt</a> </address>

<div> ... </div>

Elemento genérico de bloco. Usado para agrupar conteúdo quando não existe um elemento semântico mais adequado. É o elemento mais usado em conjunto com CSS.

<div class="caixa"> <p>Conteúdo agrupado...</p> </div>

<center>

Elemento obsoleto (deprecated). Centrava o conteúdo. Hoje em dia deve ser substituído por CSS:

<div style="text-align: center;">...</div>
