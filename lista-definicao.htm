Listas de definição

Em HTML existem três tipos de listas mais comuns:

  • Listas numeradas (<ol>)
  • Listas não numeradas (<ul>)
  • Listas de definição (<dl>)

NOTA: A lista de definição, que é ideal para criar um pequeno glossário (termo + explicação) ou uma lista de pares “conceito + descrição”.

Estrutura

Uma lista de definição usa:

  • <dl> — a lista (definition list)
  • <dt> — o termo (definition term)
  • <dd> — a definição/descrição (definition description)
<dl> <dt>HTML</dt> <dd>Linguagem de marcação usada para estruturar páginas Web.</dd> <dt>CSS</dt> <dd>Linguagem usada para definir estilos (cores, tamanhos, layout).</dd> </dl>

Nota: não é para “tabular texto à direita”; hoje isso faz-se com CSS. O objetivo do <dl> é representar termos e descrições.

Exemplo prático (glossário)

URL
Endereço de um recurso na Internet (página, imagem, ficheiro, etc.).
Link
Hiperligação que permite saltar para outra página ou secção.
Atributo
Informação extra numa tag HTML, por exemplo href em <a>.
